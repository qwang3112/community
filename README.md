<h1 style="text-align: center">community</h1>

#### 项目简介
社区论坛系统。

*  后端：Spring Boot + JPA + thymeleaf模板
*  数据库：MySQL
*  前端UI：Semantic UI框架

#### 主要特性
- 使用Jpa减少了写重复的SQL
- 前后端统一异常拦截处理
- 支持Markdown语法编辑

####  系统功能
- 注册登录：新增注册登录功能

- 博客管理：编辑，删除帖子

- 板块管理：新增，修改，删除板块

- 标签管理：新增，修改，删除标签

- 板块展示：按板块展示帖子

- 标签展示：按标签展示帖子

- 联系版主：个人信息

- 游客评论：游客可以对帖子进行评论

- 搜索：全文搜索帖子
